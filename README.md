# 🚀 Real-Time System Monitoring Dashboard

A real-time monitoring dashboard built using Python and Streamlit.

## Features
- CPU Monitoring
- Memory Monitoring
- Disk Usage Monitoring
- Live Charts
- Top Running Processes
- Real-Time Refresh

## Tech Stack
- Python
- Streamlit
- psutil
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
